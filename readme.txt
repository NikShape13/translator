Скрипт работает с помощью библиотеки googletrans, работающей с Google API. 
Берёт ваши записи из файла 'text.txt', построчно переводит и прописывает 
результат в файле 'result.txt'.

Для функционирование скрипта необходимо:
1) Установить заимствования через команду 'pip install googletrans==3.1.0a0',
2) Добавить список строк в файл 'text.py'(не забывайте их сохранять любые изменения в текстовых файлах),
3) Запустить скрипт командой 'main.py',
4) Результат появится в файле 'result.txt', будет дан перевод фразы на русский, 
а также определён язык, с которого был выполнен перевод.
